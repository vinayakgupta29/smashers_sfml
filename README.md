# Smashers SFML
SFML Game to help me fidget during my ADHD sessions.

# Requirements :

SFML 3.0+

# Installation :

### Arch

```bash
sudo pacman -S sfml base-devel
```

```code
git clone https://github.com/vinayakgupta29/smashers_sfml.git
cd smashers_sfml
```

### Mac 

#### Homebrew

```zsh
brew install gcc sfml
```
### Build 

#### Release 
```zsh
make release
```
#### Debug
```zsh
make
```

# Play

```bash
./bin/smashers

```
